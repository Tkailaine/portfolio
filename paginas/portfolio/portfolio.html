Portfólio

Advocacia

Site criado para apresentação de escritório de advocacia, mas também pode ser utilizado para apresentação de outros negócios. Seu intuito é apresentar a qualidade dos serviços prestados pela empresa e persuadir o visitante a contratá-la. Além disso, o site apresenta animações interativas passando modernidade e confiabilidade para o usuário.

  • Categorias de atuação interativas
  • Designer clean e moderno
  • Responsividade para diferentes telas
  • Formulário para contato
  • Animações especiais

Cafeteria

Site criado para apresentação de fast-foods, restaurantes, etc. Estritamente expositivo, mostrando seus produtos, cardápio e localidade. Seu intuito é atrair o usuário para visitar o estabelecimento e consumir seus serviços. Além disso, apresenta um Designer elegante e diversas páginas de navegação.

  • Páginas de navegação com exposição dos produtos
  • Cardápio
  • Responsividade para diferentes telas
  • Menu automatizado com slideshow
  • Animações especiais

Autônomo

Site criado para apresentação de serviços prestados por trabalhadores autônomos. Seu intuito é passar maior credibilidade para o público, além disso, obter maior visibilidade e fortalecer a imagem da empresa através de exposição de relatos e serviços anteriores.

  • Galeria automatizada para dispositivos móveis
  • Designer clean e moderno
  • Responsividade para diferentes telas
  • Formulário para contato
  • Animações especiais
